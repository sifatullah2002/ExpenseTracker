# Expense Tracker

## Overview

The Expense Tracker is a simple Java-based desktop application that allows users to manage their expenses. It provides functionalities for adding, viewing, searching, and saving expense entries. The application uses an SQLite database to store expense records, ensuring data persistence across sessions.

## Features

- **Add Expenses**: Users can input the category, amount, and date of each expense.
- **View All Expenses**: A table displays all recorded expenses, along with a total of all expenses.
- **Search Expenses by Date**: Users can search for expenses recorded on a specific date.
- **Save to CSV**: Users can export the expense records to a CSV file for external use.
- **Responsive GUI**: The application interface adjusts gracefully to different window sizes, ensuring a user-friendly experience.

## Technologies Used

- **Java**: The core programming language used to develop the application.
- **SQLite**: A lightweight database engine for storing expense records.
- **Swing**: Java's built-in GUI toolkit for creating the application's user interface.
